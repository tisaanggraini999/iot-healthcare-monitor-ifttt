# iot-healthcare-monitor-ifttt

This project demonstrates an end-to-end IoT healthcare monitoring system
that acquires real-time blood pressure data using ESP32 and logs the data
to Google Spreadsheet via HTTP and IFTTT Webhooks.

## System Pipeline
Blood Pressure Sensor → ESP32 → HTTP POST → IFTTT → Google Spreadsheet

## Key Features
- Real-time blood pressure acquisition
- HTTP-based cloud communication
- Serverless cloud data logging
- Healthcare-oriented IoT architecture

## Technologies
- ESP32
- C/C++
- HTTP POST
- IFTTT Webhooks
- Google Spreadsheet
